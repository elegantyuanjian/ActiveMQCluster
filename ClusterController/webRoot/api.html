Java连接过程

  • 1.解压Java接入库压缩文件
  • 2.在应用工程中，加入解压文件中的jar包
  • 3.把ClusterLinkForClient.xml文件放入工程的根目录中
  • 4.修改ClusterLinkForClient.xml中中央控制器的IP
  • 5.API使用，参考应用例子
